# Code of Conduct

## Our Pledge

We want OpenOutlier to be a welcoming, respectful project for everyone.

## Our Standards

Examples of behavior that helps build a positive environment:

- being respectful in disagreements
- assuming good intent
- giving constructive, actionable feedback
- helping new contributors get oriented

Examples of unacceptable behavior:

- harassment or intimidation
- hateful or discriminatory language
- personal attacks
- publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that violate this code of conduct.

## Scope

This applies in project spaces and public spaces when someone is representing the project.
