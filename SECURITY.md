# Security Policy

If you discover a security issue in OpenOutlier, please do not open a public issue first.

Report it privately to:

- Theo Graeser
- `theograeser@graemedias.com`

Please include:

- a description of the issue
- impact assessment if known
- reproduction steps
- any suggested mitigation

We will aim to acknowledge reports promptly and coordinate a fix before public disclosure.
